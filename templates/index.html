Pet Project to learn viewflow library with Django.

Let's imagine the IT company Software+. A lot of great developers work here. And it's important to make the process of grade improvement as easy as possible.

This project "ICrm" should help the company with this issue.

User stories
Admin user Delivery manager user Customer user Developer user

Admin user

Can add new delivery manager.

Can add new customer.

Can add new developer.

Delivery manager user

Can suggest a developer to the particular customer.

Can see all started "grade improvement" processes by his developers.

Can approve or reject a "grade improvement" process has been started by his developer.

Customer user

Can see all his developers.

Can see all "grade improvement" processes initiated by his developers.

Can approve or reject a "grade improvement" process initiated by his developer.

Developer user

Can start a "grade improvement" process (only one started process can be at a time).

Can see all initiated processes by himself (with current status).
